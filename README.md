# mcstat-cli
Stat Minecraft servers with a lightweight, pure JavaScript command line.

## Install
```bash
npm i -g mcstat-cli
```

## Usage
mcstat [options] \<address\>[:\<port\>]

### Options
-b | --bedrock - Ping a bedrock server
<br>
-s | --serverhost \<string\> - override server host
<br>
-p | --serverport \<number\> - override server port
<br>
-r | --protocolversion \<number\> - set protocol version
<br>
-v | --verbose - verbose log output
<br>
-h | --help - shows help
